FSL Evaluation and Example Data Suite (FEEDS)

There is an evaluation and example data package available to accompany FSL. This package performs two functions - it tests whether your FSL tools are working properly, and provides example data to try out FSL with.

The evaluation of FSL is carried out by a single script which normally takes between 1 and 2 hours to run (depending on your computer), testing most of the major tools in FSL on the provided example data. It compares the output of each test with precomputed output data supplied with the package, and reports significant differences as failures.

The example data includes both example input and example output data. There is a detailed description of how to analyse the input data - this is a quick way of starting to use FSL.
