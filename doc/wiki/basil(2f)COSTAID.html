COST AID Toulouse 2013 Summer School

Pre-processing Workshop

From here you may download the instructions and data to be used in the pre-processing workshop for the afternoon of the 3rd day of the summer school.

COSTAID_Toulouse.zip (32 Mb)

The workshop will require FSL (v5.0.4 or later) to be installed - please do this before the event, since there will not be time to do this on the day - installation instructions can be found here.

The workshop primarily uses the BASIL tools within FSL that are designed for the processing ASL data, it includes both command line tools and a GUI for a wide range of ASL data - more details along wiht further examples can be found from the BASIL tool menu toward the top of this page.
