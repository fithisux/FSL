FEEDS Timings Tables

These tables give the output of running /usr/bin/time ./RUN all in the top level FEEDS directory; this is user CPU time, i.e. the time with a u by it.

If you are interested in adding your timings to this table, email us with the full output of running the test, along with a description of your hardware, the output of uname -a and the version of the compiler (if any) that you used to compile FSL. Note that a timing is not very meaningful if you do not have enough RAM for the test to run without needing to do any swapping.
